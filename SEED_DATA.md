# بيانات Seed التجريبية

## التحذير

جميع البيانات التي ينشئها `assets/seed.js` تجريبية بالكامل، ولا تمثل ورشًا أو محلات أو سطحات أو مراكز صيانة حقيقية.

## المحتوى

- خمس مدن تجريبية أساسية: بريدة، عنيزة، الرس، البكيرية، المذنب.
- ثلاثة شركاء لكل نوع خدمة في كل مدينة.
- الأنواع: ورش فحص، محلات قطع غيار، سطحات، مراكز صيانة دورية.
- الإجمالي: 60 شريكًا تجريبيًا.
- كل سجل يحتوي على:
  - `isDemo: true`
  - `demoNotice`
  - اسم ينتهي بـ `(تجريبي)`
  - موقع بصياغة `موقع تجريبي داخل مدينة ...`

## عدم الاختلاق أثناء التشغيل

المطابقة لا تنسخ الشريك إلى مدينة أخرى، ولا تغير رقمه أو مسافته أو توفره. إذا لم توجد بيانات Seed مطابقة، تظهر حالة «لا يوجد تطابق حاليًا».

## حساب الشريك

```text
رمز الشريك: WA-DEMO
الرمز السري: 1234
```

## سجل مالي تجريبي

ينشأ طلب وإحالة ورسم واحد لتوضيح لوحة الشريك. لا يمثل أي مطالبة أو دفعة حقيقية.
