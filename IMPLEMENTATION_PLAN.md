# خطة التنفيذ المنجزة

## 1. تثبيت النسخة المرجعية

- فك آخر حزمة حالية.
- مقارنة الملفات مع النسخة السابقة.
- تحديد الملفات المتأثرة فقط دون إعادة بناء المشروع.

## 2. ترتيب رحلة العميل

- نقل التواصل بعد الخصم.
- إزالة إرسال رابط الطلب عبر WhatsApp.
- وضع رابط الطلب والبديل في نهاية النتيجة والمتابعة.

## 3. تحسين التخطيط العام

- إضافة الرئيسية والسابق في التخطيط المشترك.
- إضافة إرشاد خاص بكل صفحة.
- تحسين الحركات والمسافات والبطاقات والأزرار للجوال وRTL.

## 4. تخصيص انضمام الشريك

- بناء ساعات أسبوعية مستقلة.
- فصل وثائق السطحة.
- تخصيص الصيانة وقطع الغيار والورش.
- تطبيق شروط الخصومات وتواريخها.

## 5. تدقيق لوحة الشريك

- مراجعة الإحالات والحالات والتقييمات والاعتراضات.
- ربط الخصومات والفواتير والمدفوعات والملف.
- إصلاح منع تكرار جلسة بيانات الدخول.

## 6. الاختبار والتسليم

- فحص تركيب JavaScript.
- اختبارات منطق البيانات والعلاقات.
- تدقيق HTML والوصولية والروابط.
- اختبار متطلبات التعديل المستهدفة.
- اختبار HTTP لجميع الصفحات والأصول.
- إنشاء الحزمة وملفات التوثيق والكود الكامل للملفات المتغيرة.
