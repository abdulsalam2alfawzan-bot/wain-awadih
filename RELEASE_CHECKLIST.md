# قائمة تحقق الإصدار

## مكتمل في نسخة العرض

- [x] المسارات الأربعة تعمل بالبيانات المحلية.
- [x] المسار الرئيسي واضح في الصفحة الأولى.
- [x] الموديلات مرتبطة بالشركة.
- [x] الأعراض الخطرة لها توجيه آمن.
- [x] لا أسعار تقديرية أو تشخيص نهائي.
- [x] الطلب والإحالة يسبقان عرض الشريك.
- [x] شريك واحد كل مرة وثلاث إحالات كحد أقصى.
- [x] publicToken لا يحتوي الجوال.
- [x] رابط واتساب جاهز.
- [x] التأكيد والتقييم والبدائل تعمل.
- [x] الرسم عند الوصول وبدء الاستقبال فقط.
- [x] المطالبات الشهرية وحد 100 ريال يعملان.
- [x] الاعتراضات والخصومات والمدفوعات التجريبية تعمل.
- [x] Seed موسوم بأنه تجريبي.
- [x] لا روابط محلية ميتة.
- [x] RTL وMobile-First وCSP.
